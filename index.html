A growing library of production-ready n8n workflows, written as guided case studies for the AI Marketing Lab at IE Business School. Each workflow ships with the raw JSON you can import, a plain-language anatomy of every node, and a step-by-step build from scratch so you can rebuild it yourself and understand the craft.

00
How this library works

One workflow per guide. Two ways to use each.

Every guide follows the same four-section shape so you always know where to look: an overview of what the workflow does, the prerequisites you'll need, a quick-start import path, and a detailed node-by-node anatomy. An optional "build from scratch" section walks through each node's configuration if you'd rather learn by rebuilding.

📦
The fast path

Import the JSON

Each guide links a downloadable .json file. Drop it into n8n → Import from File, replace the REPLACE_WITH_* placeholders with your own IDs and tokens, link credentials, run.

🛠️
The slow path

Build from scratch

Prefer to learn the n8n UI node by node? Each guide ends with a step-by-step rebuild — same outcome, but you'll know exactly what each expression and credential does.

🎓
For the classroom

Built for non-technical profiles

No prior coding. Concepts are introduced when they're needed and nowhere else. Every external account used has a free tier and is covered in the prerequisites section.
